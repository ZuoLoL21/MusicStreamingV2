Database 
------------------------
User
- uuid 
- username
- email
- hashed_password
- bio
- profile_image_path
- created_at
- updated_at
Primary Key uuid

Artist
- uuid
- user_uuid User (nullable for legacy/deceased artists)
- artist_name
- bio
- profile_image_path
- created_at
- updated_at
Primary Key uuid
Index user_uuid
Unique artist_name

ArtistMember
- uuid
- artist_uuid Artist
- user_uuid User
- role enum('owner', 'manager', 'member')
- joined_at
Primary Key uuid
Unique (artist_uuid, user_uuid)
Index artist_uuid
Index user_uuid

Music
- uuid
- from_artist Artist
- uploaded_by User
- in_album Nullable[Album]
- song_name
- created_at
- updated_at
- path_in_file_storage
- play_count
- duration_seconds
Primary Key uuid
Index from_artist

Likes
- uuid
- from_user User
- to_music Music
Primary Key uuid
Unique (from_user, to_music)
Index to_music
Index from_user

Follows
- uuid
- from_user User
- to_user Nullable[User]
- to_artist Nullable[Artist]
- created_at
Primary Key uuid
Index from_user
Index to_artist
Check: (to_user IS NULL) != (to_artist IS NULL)  -- exactly one must be set


TagAssignment
- uuid
- music_uuid Music
- tag_name MusicTags
- created_at
Primary Key uuid
Unique (music_uuid, tag_name)
Index music_uuid

MusicTags
- tag_name
- tag_description
- created_at
Primary Key tag_name

Album
- uuid
- from_artist Artist
- original_name 
- description 
- created_at
- updated_at
Primary Key uuid
Unique (from_artist, original_name)
Index from_artist

Playlist
- uuid 
- from_user User 
- original_name
- description
- is_public
- created_at
- updated_at
Primary Key uuid
Unique (from_user, original_name)
Index from_user

PlaylistTrack
- uuid
- music_uuid
- position
- playlist_uuid
- added_at
Primary Key uuid
Index (playlist_uuid, position)

ListeningHistory
- uuid
- user_uuid User
- music_uuid Music
- played_at timestamp
- listen_duration_seconds 
- completion_percentage
Primary Key uuid
Index user_uuid
